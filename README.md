Dự án: Ứng dụng quản lý sức khỏe thú cưng

Mô tả dự án: PetHealth là một ứng dụng di động được nhóm xây dựng nhằm hỗ trợ người
nuôi thú cưng quản lý toàn bộ thông tin sức khỏe của thú cưng một cách đơn giản,
khoa học và hiện đại. Trong bối cảnh ngày càng nhiều gia đình nuôi thú cưng
nhưng việc ghi nhớ lịch tiêm phòng, theo dõi hồ sơ sức khỏe hoặc quản lý nhiều
thú cưng vẫn gặp nhiều khó khăn, ứng dụng PetHealth giúp người dùng giải quyết
các vấn đề đó trong một hệ thống duy nhất.

Tính năng chính:
- Quản lý tài khoản người dùng.
- Quản lý thú cưng.
- Theo dõi sức khỏe.
- Sổ tiêm & thuốc điện tử.
- Nhắc lịch & thông báo.
- Quản lý thông báo.

Hướng dẫn tải dự án:
Dự án được lưu trữ tại GitHub: https://github.com/sangg05/PET_H
EALTH.git
Cách 1: Tải trực tiếp (Download ZIP)
1. Truy cập link dự án.
2. Nhấn Code → Download ZIP.
3. Giải nén file và mở bằng Android Studio.
Cách 2: Clone bằng Git
1. Mở CMD/Terminal.
2. Di chuyển đến thư mục muốn lưu.
3. Chạy lệnh:
git clone https://github.com/sangg05/PET_HEALTH.git
4. Mở thư mục dự án trong Android Studio và chạy ứng dụng.
